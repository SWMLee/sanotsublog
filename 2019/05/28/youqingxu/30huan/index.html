高中印象无病呻吟

欢

2019-05-28

昨天是会考，也是农历五月二十三，堂姐生日。说实在的，唯一一个能编上号的姐，却因为太多原因，已经很久没见过面了。倒是她20岁办生回来过一次，因为怕是回来被放人户所以不想回来了。只是有点事情必须地说一下，那就是她已经长大了。

今天，却是我的生日，是的，老妈提醒我的。前几天打电话来说以至于我现在都还记得。第一次在读书期间过生，可怜却也只有像平常一样，什么事也没得做。前天打篮球倒是把腿蹭掉了几块皮。这倒不是大事，只是这个事让我痛了些，今大早起来又打球，以致才好的疤又打掉了。其实也是好的，因为至少可以明白今天还是有乱动的。天也热，坐在最角落，干不得的大事。

重点是这个问题，老妈说她已经8个生日没和我过了，这样到还是给了我一个念头，因为至少还是有着机会的。可这几天却真的不高兴了，我觉得有什么事要发生。

必须地，我想去看一看，到那里去一趟，就是最后一趟了，这就快结束了。人生就这么空旷而无聊。十二天磨一剑，就这样草草了结，马上就是我们的葬所了。

她让我说，我不太想说因为很难说出口，这是一个问题。譬如，我站在门后，你却破门进来。

期末考试之后，开学之前，我想我至少还有两件事可以做，一是娬嫋（wuniao）嫣。这是件大事，我一直想的事，因为太草草，必须要客观解决。至于启示录，有机会再写。时间是一点一点出来的，这是时间的不完整性，自然也有他的完全性，完全地一去不回来。

我还是想睡觉，这像现在在写的作文，一个羸比一个，完全没有了风格，没有了感情。对自己是否一点作用都没有了，这是一个问题。至于需不需要自我难过一下，这才是大事。今天我生日，是非常无聊地过一天，一天又一天，什么都没了，什么都完了。

为了期末考试，再努力一下吧。
